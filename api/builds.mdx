## Get Build

<CodeGroup>
```bash GET /api_3/build/{build_id}
curl -X GET \
  "https://copr.fedorainfracloud.org/api_3/build/12345"
```
</CodeGroup>

Get details for a single Copr build.

### Path Parameters

<ParamField path="build_id" type="integer" required>
  Build ID
</ParamField>

### Response

<ResponseField name="id" type="integer">
  Build ID
</ResponseField>

<ResponseField name="state" type="string">
  Build state: "importing", "pending", "starting", "running", "succeeded", "failed", "canceled", "skipped"
</ResponseField>

<ResponseField name="projectname" type="string">
  Project name
</ResponseField>

<ResponseField name="ownername" type="string">
  Project owner name
</ResponseField>

<ResponseField name="repo_url" type="string">
  Repository URL for DNF/Yum
</ResponseField>

<ResponseField name="source_package" type="object">
  Source package information
  <Expandable title="properties">
    <ResponseField name="name" type="string">
      Package name
    </ResponseField>
    <ResponseField name="version" type="string">
      Package version
    </ResponseField>
    <ResponseField name="url" type="string">
      SRPM download URL
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="submitted_on" type="integer">
  Unix timestamp when build was submitted
</ResponseField>

<ResponseField name="started_on" type="integer">
  Unix timestamp when build started
</ResponseField>

<ResponseField name="ended_on" type="integer">
  Unix timestamp when build ended
</ResponseField>

<ResponseField name="submitter" type="string">
  Username of build submitter
</ResponseField>

<ResponseField name="chroots" type="array">
  List of chroot names where build is executed
</ResponseField>

<ResponseField name="is_background" type="boolean">
  Whether this is a background build
</ResponseField>

## List Builds

<CodeGroup>
```bash GET /api_3/build/list
curl -X GET \
  "https://copr.fedorainfracloud.org/api_3/build/list?ownername=@copr&projectname=copr-dev&limit=20"
```
</CodeGroup>

List all builds in a Copr project with pagination and filtering.

### Query Parameters

<ParamField query="ownername" type="string" required>
  Project owner name
</ParamField>

<ParamField query="projectname" type="string" required>
  Project name
</ParamField>

<ParamField query="packagename" type="string">
  Filter by package name
</ParamField>

<ParamField query="status" type="string">
  Filter by build status (succeeded, failed, canceled, etc.)
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Number of items per page
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of items to skip
</ParamField>

### Response

<ResponseField name="items" type="array">
  Array of build objects (same structure as Get Build)
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata
</ResponseField>

## Create Build from URL

<CodeGroup>
```bash POST /api_3/build/create/url
curl -X POST \
  "https://copr.fedorainfracloud.org/api_3/build/create/url" \
  -H "Content-Type: application/json" \
  -d '{
    "ownername": "myuser",
    "projectname": "my-project",
    "pkgs": "https://example.com/package.src.rpm",
    "chroots": ["fedora-40-x86_64"]
  }'
```
</CodeGroup>

Create a build from SRPM URL(s). Requires authentication.

### Request Body

<ParamField body="ownername" type="string" required>
  Project owner name
</ParamField>

<ParamField body="projectname" type="string" required>
  Project name
</ParamField>

<ParamField body="pkgs" type="string" required>
  SRPM URL(s), one per line for multiple packages
</ParamField>

<ParamField body="chroots" type="array">
  List of chroot names to build in (defaults to all project chroots)
</ParamField>

<ParamField body="background" type="boolean" default="false">
  Run as background build
</ParamField>

<ParamField body="timeout" type="integer">
  Build timeout in seconds
</ParamField>

<ParamField body="enable_net" type="boolean">
  Enable network access during build
</ParamField>

### Response

<ResponseField name="items" type="array">
  Array of created build objects
</ResponseField>

---

## Create Build from Upload

<CodeGroup>
```bash POST /api_3/build/create/upload
curl -X POST \
  "https://copr.fedorainfracloud.org/api_3/build/create/upload" \
  -F "ownername=myuser" \
  -F "projectname=my-project" \
  -F "pkgs=@/path/to/package.src.rpm" \
  -F "chroots=[\"fedora-40-x86_64\"]"
```
</CodeGroup>

Create a build from an uploaded SRPM file. Requires authentication.

### Request Body (multipart/form-data)

<ParamField body="ownername" type="string" required>
  Project owner name
</ParamField>

<ParamField body="projectname" type="string" required>
  Project name
</ParamField>

<ParamField body="pkgs" type="file" required>
  SRPM file to upload
</ParamField>

<ParamField body="chroots" type="array">
  List of chroot names to build in
</ParamField>

### Response

Returns the created build object with 200 status code.

---

## Create Build from SCM

<CodeGroup>
```bash POST /api_3/build/create/scm
curl -X POST \
  "https://copr.fedorainfracloud.org/api_3/build/create/scm" \
  -H "Content-Type: application/json" \
  -d '{
    "ownername": "myuser",
    "projectname": "my-project",
    "clone_url": "https://github.com/user/repo.git",
    "committish": "main",
    "subdirectory": "",
    "spec": "package.spec",
    "source_build_method": "rpkg",
    "chroots": ["fedora-40-x86_64"]
  }'
```
</CodeGroup>

Create a build from a source code management system (Git, SVN). Requires authentication.

### Request Body

<ParamField body="ownername" type="string" required>
  Project owner name
</ParamField>

<ParamField body="projectname" type="string" required>
  Project name
</ParamField>

<ParamField body="clone_url" type="string" required>
  SCM repository URL
</ParamField>

<ParamField body="committish" type="string">
  Branch, tag, or commit hash (defaults to default branch)
</ParamField>

<ParamField body="subdirectory" type="string">
  Subdirectory within repository
</ParamField>

<ParamField body="spec" type="string">
  Path to .spec file within repository
</ParamField>

<ParamField body="scm_type" type="string">
  SCM type: "git" or "svn" (auto-detected from URL)
</ParamField>

<ParamField body="source_build_method" type="string">
  SRPM build method: "rpkg", "tito", "tito_test", "make_srpm"
</ParamField>

<ParamField body="chroots" type="array">
  List of chroot names to build in
</ParamField>

### Response

Returns the created build object with 200 status code.
